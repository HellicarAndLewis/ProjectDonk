Detailed Description

This calls glEnable/disable while the object is in scope, and then returns openGL to its previous state when the destructor is called.

e.g. { // enable depth test ScopedGLCapability depth(GL_DEPTH_TEST, true);

// do some stuff } // depth is out of scope, so GL_DEPTH_TEST should be reverted to state previously set.
